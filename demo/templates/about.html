What is Imprint?

Imprint is a lightweight distributed tracing solution that helps you understand how requests flow through your application stack, from browser to backend.

How It Works

  • The Browser Agent automatically instruments page views and fetch requests
  • Each request includes a traceparent header following W3C Trace Context
  • The Go Agent extracts this context and creates correlated spans
  • Both agents batch and send spans to the ingest endpoint
