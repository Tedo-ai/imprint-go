What's happening?

            
    
                
  • The Browser Agent creates a trace when you click a button
    • 
                
  • The traceparent header is injected into fetch requests
    • 
                
  • The Go Agent extracts the trace context and creates child spans
    • 
                
  • Both agents send spans to the ingest endpoint
    •
